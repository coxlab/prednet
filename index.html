PredNet

Code accompanying "Deep Predictive Coding Networks for Video Prediction and Unsupervised Learning"

Bill Lotter, Gabriel Kreiman, and David Cox (2015)

View on arXiv View on GitHub Download .zip Download .tar.gz

The PredNet is a deep convolutional recurrent neural network inspired by the principles of predictive coding from the neuroscience literature [1, 2]. It is trained for next frame video prediction with the belief that prediction is an effective objective for unsupervised (or "self-supervised") learning [e.g. 3-11]. Next frame predictions on the Caltech Pedestrian [12] dataset are shown below. The model was trained on the KITTI dataset [13].
